# PageFrame (structure)

How pages are **built**: registry metadata, template names, validators, and expansion **patterns** — not the runtime motion (that’s **DriveShaft**).

- **`registry/`** — `page-types.ts` bridge, `templates.ts`, `validators.ts`
- **`expansion/`** — slug patterns, snowball / hub growth hooks

Cross-ref: **`100KPageMaker/page-types/`** and **`100KPageMaker/versions/`** for versioned schema snapshots.
